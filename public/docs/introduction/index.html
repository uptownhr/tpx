Getting Started with Jien Framework

How to Install

First, grab the source code from github

http://github.com/jaequery/jien

Next ...

  1. Extract

    If your public folder is: /home/jaequery/domains/bla.com/public, the the code should be placed at /home/jaequery/domains/bla.com

  2. Confirm

    Once extracted, check that your folder contains these:

    • /home/jaequery/domains/bla.com/application
    • /home/jaequery/domains/bla.com/library
    • ... and make sure your cache and sessions folder have write permissions

  3. Configure

    Edit /home/jaequery/domains/bla.com/application/configs/application.ini

    Modify accordingly to your needs:

    ... db settings, disqus/twitter id, and memcache(optional)

  4. Populate the database

    Load the sql dump file which is located at:

    ... /home/jaequery/domains/bla.com/sql/jien.sql

You are all setup

Verify everything works by going to http://yoursite/admin

user: admin
pass: demo

Troubleshoot

If you still have issues, make sure you have properly enabled mod_rewrite ony your webserver.
